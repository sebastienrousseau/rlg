Macro rlg::macro_error_log

source · 
macro_rules! macro_error_log {
    ($time:expr, $component:expr, $description:expr) => { ... };
}
Expand description

This macro creates an ERROR level log entry with a default session ID and format. The session ID is generated randomly and the log format defaults to CLF.

§Parameters

  • time: The timestamp of the log entry.
  • component: The system component that generated the log.
  • description: A textual description of the log event.

§Example

use rlg::{macro_error_log, macro_log};
use rlg::log_level::LogLevel;
use rlg::log_format::LogFormat;
let log = macro_error_log!("2024-08-29T12:00:00Z", "Database", "Connection failed");

Usage: let log = macro_error_log!(time, component, description); Macro for error log with default session id and format
